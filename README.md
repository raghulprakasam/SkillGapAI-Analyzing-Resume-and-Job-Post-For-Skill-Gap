# SkillGapAI-Analyzing-Resume-and-Job-Post-For-Skill-Gap
he SkillGapAI system analyzes a Resume and a Job Description (JD) to quantify the difference in required vs. existing skills. It uses NLP and vectorization to accurately identify and weigh skills from both documents, resulting in a Match Score and a detailed report of the skill gaps, which informs training recommendations.
